# photoRedactor

Веб-приложение помогает подготовить фотографии для печатного альбома: автоматически вписывает снимки в заданный формат страницы, оставляет свободное поле под текст и формирует готовый архив для типографии.

## Что умеет
- Поддерживает загрузку нескольких изображений (JPG, PNG) через выбор файлов или drag-and-drop.
- Настраивает параметры страницы: ширина/высота в мм, DPI печати, отступы и доля пространства под текст.
- Визуально показывает подготовленные страницы: фото вписано в рамку, внизу выделена зона для подписей.
- Сохраняет каждую страницу в JPEG и собирает их в ZIP-архив («Отправить на печать»).

## Как запустить
1. Откройте `index.html` в любом современном браузере (дополнительный сервер не требуется).
2. Укажите параметры альбома.
3. Загрузите фотографии и нажмите «Подготовить фото» — увидите превью.
4. Нажмите «Отправить на печать», чтобы скачать архив `album-ready.zip` со всеми подготовленными JPEG.

## Запуск на macOS
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/<your-account>/photoRedactor.git
   cd photoRedactor
   ```
2. Запустите локальный сервер (так удобнее, чем открывать файл напрямую):
   ```bash
   python3 -m http.server 8000
   ```
   Затем откройте в браузере ссылку [http://localhost:8000](http://localhost:8000) и выберите `index.html` в списке файлов.
3. При необходимости можно открыть `index.html` двойным кликом — приложение тоже работает, но сервер удобнее для стабильной загрузки файлов в разных браузерах.

## Как обновлять репозиторий на компьютере
1. Убедитесь, что находитесь в каталоге проекта: `cd /path/to/photoRedactor`.
2. Проверьте текущее состояние (есть ли незакомиченные изменения):
   ```bash
   git status
   ```
3. Обновите код из основной ветки удалённого репозитория:
   ```bash
   git pull origin main
   ```
4. Если работаете в другой ветке, сначала переключитесь на неё (`git checkout <branch>`), затем обновите (`git pull`). При конфликтах Git подскажет, какие файлы нужно поправить.
